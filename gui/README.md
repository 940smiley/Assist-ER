# Assist-ER Pro GUI

This GUI is built with NeutralinoJS and provides:
- Authentication panel
- Repository browser
- PR dashboard
- Workflow dashboard
- Settings panel
- Live logs panel

## Run

```bash
# install Neutralino CLI globally once
npm i -g @neutralinojs/neu
cd gui
neu run
```

## Build Windows EXE

```bash
neu build --release
```
