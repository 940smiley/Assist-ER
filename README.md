# Assist-ER is a github app that a user can install and choose all or select repos to connect to and then from the assist-er repo can clone the source code and use assist-er locally on their machine to clone repositories apply edits in batchs rather then have to manually copy and paste, assist-er also can generate workflows based on a users needs/wants/requests, able to generate documentation, able to apply changes to settings at a repo level not account level, the assist-er can also be setup to provide triage alongside dependabot or any other bot you may have installed and set up, you can assign the assist-er to a repo and give full control (merge prs, create prs, triage for issues and bugs, apply edits to files, update dependencies, revise entire codebase, create and manage settings and access tokens, secrets, variables ,gh pgaes) or select few if full control isnt something for you.
