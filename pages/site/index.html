Assist-ER

Budget + Pro GitHub maintenance platform

Features

  • Budget CLI for lightweight automation
  • Pro GUI with dashboards and live logs
  • Autonomous maintenance engine (`assist-er --pro work`)
  • Workflow repair, dependency updates, and PR processing

CLI Examples

assist-er --budget repos
assist-er --pro work
assist-er --pro repos workflows owner repo --fix

Install

pip install -e .[dev]
assist-er --help

Architecture

core/auth, core/api, core/repos, core/prs, core/branches, core/workflows, core/ai, core/utils, plugins, gui, pages, build.
